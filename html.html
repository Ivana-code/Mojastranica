HTML5 ( Hyper Text Markup Language ) je jezik koji se koristi za kreiranje dokumenta na World Wide Web-u. Nastao je uprošćavanjem SGML ( Standard Generalized Markup Language ) standarda sa vrhom opisa dokumenta koji se objavljuje na vebu. Osnivač HTML je W3C, World Wide Web Consortium na čelu sa Tim Bernerom-Lijem.


HTML logo

 Html se sastoji iz dva osnovna dijela:

    head 
        ( sadrži opis dokumenta )
    body   
        ( sadrži informacije koje će se prikazati na Web stranici ).

HEAD - zaglavlje sadrži obavezno TITLE - naslov.

Osnove HTML-a predstavljaju tagovi i atributi.
