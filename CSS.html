CSS

CSS (Cascading Style Sheets) je jezik pomoću kog se definiše izgled elemenata veb-stranice.

CSS3 je objavljen kao zvanična verzija CSS-a 12. maja 1998., čiji je razvoj započet 1998. godne i još uvijek se razvija.

Svaki opis se sastoji od tri elementa:

  1. definicija ciljnih elemenata
  2. svojstva
  3. vrijednosti

CSS se može dodati u HTML na slijedeće načine:

  • navodeći HTML tag ciljnih elemenata
  • navodeći klasu elemenata
  • navodeći direktnu identifikacionu vrijednost (ID) elementa

Ciljni elementi se definišu na tri načina:

  • u liniji /INLINE CSS/ - korisiti se STYLE atribut u HTML elementu
  • interni /INTERNAL CSS - koristeći <STYLE> element u HEAD sekciji
  • eksterni /EXTERNAL CSS/ - koriteći CSS fajla.
